Hur slutar man hata sig själv?

Att sluta hata sig själv sker sällan i ett enda skifte. Ofta börjar det i att upptäcka hur normaliserad den hårda rösten har blivit.

Var börjar man?

  • Lägg märke till hur du talar till dig själv
  • Separera tanke från sanning
  • Försök minska det som ständigt matar självföraktet

Varför sitter det så djupt?

Självhat är ofta inte bara en känsla. Det kan vara ett invant system av kritik, skam och jämförelse som pågått så länge att det känns som din personlighet.

Vad hjälper på sikt?

  • Mer ärlighet, mindre självstraff
  • Små handlingar där du inte överger dig själv
  • Att sluta kräva att värde först måste bevisas

Relaterade tillstånd

  • Varför hatar jag mig själv?
  • Varför känner jag mig värdelös?
  • Hur får man bättre självkänsla?

Att sluta hata sig själv börjar inte alltid med kärlek. Ibland börjar det bara med att inte fortsätta slå.
